# Comment Playbook — Карта топлива

Правила для комментариев в YouTube, VK, Drive2, Telegram и личных сообщений блогерам.

## Главный принцип

**Сначала польза.** Комментарий должен работать и без ссылки: совет, лайфхак, уточняющий вопрос. Ссылка — опциональное дополнение, не цель.

## Обязательные правила

1. **Be helpful first** — отвечай на боль автора/зрителя: очередь, пустая колонка, неизвестность по ДТ в области.
2. **Link only when relevant** — ссылку даём, если человек явно ищет «где заправиться» или спрашивает про карты/сервисы. В спорах о политике, в мемах, под нерелевантными роликами — без ссылки.
3. **No copy-paste spam** — каждый комментарий уникален. Шаблоны из `social/` — только отправная точка, не финальный текст.
4. **Vary text** — чередуй длину, тон, структуру: вопрос / совет / личная история / уточнение региона.
5. **Disclose «пользуюсь сам»** — честно: «нашёл карту, сам отмечаю заправки», «пользуюсь пару недель», не «официальный представитель».

## Тон по каналам

| Канал | Тон | Длина |
|-------|-----|-------|
| YouTube | разговорный, без канцелярита | 2–5 предложений |
| VK (региональные группы) | «свои», без продажи | 3–6 предложений |
| Drive2 | опыт владельца авто, технично по делу | 4–8 предложений |
| Telegram | коротко, по сути | 1–3 предложения |

## Формулировки — можно

- «Перед выездом смотрю, где недавно отмечали 95-ю»
- «Водители сами обновляют — не идеально, но лучше ехать вслепую»
- «Пользуюсь сам, когда езжу между городами»
- «Если кто-то в [регион] — отметьте свою заправку, поможет остальным»
- «Не знаю, насколько у вас актуально, у нас в области вчера так же было»

## Формулировки — нельзя

- «ЛУЧШАЯ КАРТА!!!», «СКАЧИВАЙТЕ СРОЧНО»
- «100% точные данные», «официальный источник»
- Один и тот же текст под 10 видео подряд
- Ссылка в каждом комментарии подряд с одного аккаунта
- Критика конкурентов, политические споры ради охвата
- Выдуманные цены и «инсайды» с заводов

## Когда давать ссылку

| Ситуация | Ссылка |
|----------|--------|
| «Где посмотреть, где есть бензин?» | Да, мягко |
| «Очередь на заправках в [город]» | Да, как один из вариантов |
| Новость без вопроса водителей | Нет |
| Видео про ремонт / тюнинг | Нет |
| Уже 3+ комментария про сервисы в ветке | Нет, лучше уточнить детали без ссылки |

## Структура «helpful + soft link»

1. Признать проблему («да, у нас тоже длинные очереди на трассу»)
2. Конкретный совет («смотрю свежие отметки за последние 2–4 часа»)
3. Личное («пользуюсь сам»)
4. Ссылка одной строкой, без UTM-простыней в комментарии

## Модерация и риски

- В VK-группах с жёсткой модерацией — сначала комментарий без ссылки; ссылку в ответ на вопрос
- YouTube часто прячет комментарии со ссылками — дублируй название «Карта топлива» для поиска
- Drive2 — уважай правила раздела; не оффтопить в ветки не про топливо

## Чеклист перед публикацией

- [ ] Текст уникален (не совпадает с queue за последние 7 дней)
- [ ] Есть польза без ссылки
- [ ] Указан регион/тип топлива, если уместно
- [ ] Есть «пользуюсь сам» или аналог
- [ ] Нет агрессивной рекламы и капслока
- [ ] Актуальность проверена (новости за сегодня/вчера)
