# Кодекс этики и кризисный режим

## Миссия в одном предложении

В момент дефицита топлива мы — **нейтральный источник правды для водителя**, а не витрина для сетей АЗС.

## Пять принципов

1. **Данные о наличии — общественное благо**, не товар. Никогда не продаём доступ к статусу «есть / нет».
2. **Платящая АЗС не влияет на статус наличия** — только на метку «Партнёр», цену, акции, verified feed.
3. **Репутация не продаётся** — нельзя купить «доверенный репортёр» или приоритет отчёта на карте.
4. **Прозрачность источника** — каждый статус: crowdsourced / подтверждено сетью / устарело (TTL).
5. **Кризис — режим служения**, не режим монетизации на карте.

## Кризисный режим (Crisis Mode)

### Триггеры (любой из)

- >30% АЗС в регионе со статусом «нет» по основному виду топлива (95 или ДТ)
- Официальное ограничение продаж в субъекте РФ
- Ручной переключатель владельца продукта

### Автоматические действия

| Действие | Статус |
|----------|--------|
| Реклама на карте | **ВЫКЛ** |
| Sponsored / paid pins | **ВЫКЛ** |
| Upsell Premium на карте и в боте при поиске | **ВЫКЛ** |
| Баннеры в критическом UI-потоке | **ВЫКЛ** |
| Усиленная модерация фейков | **ВКЛ** |
| Бесплатный доступ ко всем данным | **ВКЛ** (как всегда) |

### Что остаётся разрешённым в кризис

- Premium **только** для уже подписанных пользователей (push, маршруты) — без новых агрессивных продаж
- B2B **Crisis Broadcast** от **верифицированной** сети — как служба информирования, не как реклама «купите у нас»
- Донаты — ненавязчиво, не на карте

### Коммуникация в кризис

- Публичное сообщение: «Мы на стороне водителей. Карта бесплатна. Реклама отключена.»
- Партнёрства подавать как «официальное подтверждение наличия», не «рекламу сети»

## Модерация данных

| Параметр | Значение |
|----------|----------|
| TTL отчёта (бензин, кризис) | 45–60 мин |
| TTL отчёта (ДТ, стабильный период) | 2–4 ч |
| Скрытие после негативных подтверждений | 3+ |
| Штраф репутации за фейк | −50 баллов |
| Бан | повторный фейк или координированный спам |

## Антипаттерны (запрещено навсегда)

- Paywall на наличие топлива
- Платный приоритет «есть бензин» на карте
- Скрытая реклама без маркировки и erid
- Продажа геотреков пользователей
- Сокрытие конкурентов для платящей сети
- Платные «подтверждения» от АЗС без API-верификации

## Чеклист перед любым monetization-релизом

- [ ] Не затрагивает бесплатный доступ к данным о топливе
- [ ] Есть маркировка «Реклама» + erid (если реклама)
- [ ] Crisis Mode протестирован (feature flag)
- [ ] Юридическая проверка (`legal-checklist.md`)
- [ ] MAU ≥ порога из `revenue-milestones.md`

## Эскалация

При сомнении — **не запускать**. Спросить владельца продукта. В кризис приоритет: доверие > MRR.
