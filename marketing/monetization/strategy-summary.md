# Стратегия монетизации — краткая сводка

> **Статус:** подготовка. Активация дохода — после **10 000 MAU** (2 мес. подряд) + accuracy >75%.  
> Полный анализ: см. историю стратегии в чате fuel-map (июнь 2026).

## Одна фраза

«Waze для топлива в России: бесплатны для водителей в кризис, зарабатываем на сетях АЗС, которым нужен трафик и правда о спросе».

## Три опоры

| # | Опора | Доля выручки (цель) |
|---|-------|---------------------|
| 1 | **B2B SaaS** для сетей АЗС | 50–60% |
| 2 | **Premium** для водителей (комфорт, не доступ) | 15–25% |
| 3 | **Партнёрства + data** (с 50k–100k MAU) | 20–35% |

## Принципы (не обсуждаются)

1. **Данные о наличии топлива — бесплатно навсегда.** Карта, отчёты, базовый поиск, 3 push/день в боте.
2. **B2B first.** Пилоты с независимыми сетями → кейс → ВИНК.
3. **Premium 149 ₽/мес** — только push, маршруты, офлайн, мультигород, без рекламы в ленте. **Не** продаём доступ к информации о топливе.
4. **Кризисный режим** — автоматически: без рекламы на карте, без paid pins, без upsell Premium на карте.

## Что НЕ делать

- Paywall на «есть / нет топлива»
- Платный приоритет статуса АЗС
- Скрытая реклама без erid
- Продажа геолокации пользователей
- Блокировка конкурентов для платящей сети

## B2B — что продаём

Не «рекламу», а **операционную аналитику и управление спросом**:

- Heatmap спроса, очереди, отток к конкурентам
- Verified Status — официальный feed от сети + бейдж
- Crisis Broadcast — push «появился ДТ» от верифицированного источника
- Franchisee Pack для джобберов и независимых сетей

## Premium — что продаём

| Бесплатно навсегда | Premium (149 ₽) |
|--------------------|-----------------|
| Карта и отчёты | Мгновенные push (без лимита) |
| Базовый поиск | Маршрут «заправка по пути» |
| 3 уведомления/день в боте | До 5 городов, офлайн-кэш |
| — | Фильтр «только подтверждённые» |
| — | Без рекламы в ленте |

## Прогноз выручки (консервативно)

| MAU | MRR |
|-----|-----|
| 10k | 130–230 тыс. ₽ |
| 50k | 1,2–2,2 млн ₽ |
| 100k | 3,8–6,5 млн ₽ |

## Следующий шаг после 10k

1. Premium waitlist → мягкий запуск 49 ₽ промо  
2. Пилот 30 дней с 2 независимыми сетями в одном регионе  
3. Кейс с цифрами → питч ЛУКОЙЛ / Газпромнефть региональным дирекциям  

## North Star до монетизации

% отчётов, подтверждённых вторым пользователем за 15–30 мин — не установки и не MRR.
