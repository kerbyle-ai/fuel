# Партнёрство: автошколы

> **Статус:** черновик one-pager. Запуск пилота — при стабильном покрытии в городе (не обязательно 10k MAU, достаточно точности в районе автошколы).

## Для кого

- Сети автошкол (3+ филиала в городе)
- Крупные учебные центры с собственным fleet учебных авто
- Онлайн-школы с практикой в одном регионе

## Боль партнёра

| Проблема | Влияние |
|----------|---------|
| Инструктор с учеником кружит в поисках 95 | Потеря часа практики, негатив, плохие отзывы |
| Ученики спрашивают «где заправиться» — нет единого ответа | Размытие бренда школы |
| Дефицит топлива — отмена занятий | Возвраты, репутационные риски |

## Оффер «Автошкола + Карта топлива»

### Для школы (бесплатно на пилоте)

- **Co-branded landing:** «Наши ученики знают, где сейчас есть бензин — [Название школы] × Карта топлива»
- **QR в классе и на парковке:** ссылка на PWA с UTM `autoschool_[slug]`
- **Методичка для инструкторов** (1 стр.): как пользоваться картой на занятии
- **Отчёт для администратора:** сколько учеников воспользовались (агрегат, без ПДн)

### Для учеников

- Карта бесплатно  
- **Промокод на Premium:** первый месяц **0 ₽** или **49 ₽** при записи в школу (лимит [N] кодов/мес)

### Монетизация для нас (после пилота)

| Модель | Условие | Цена |
|--------|---------|------|
| **Спонсорство раздела «Ученикам»** | Лого + QR в 3+ филиалах | 3 000–8 000 ₽/мес за город |
| **Affiliate Premium** | 20% с подписки ученика 3 мес. | performance |
| **Пакет для сети (10+ филиалов)** | Единый бренд + отчёты | 15 000–25 000 ₽/мес |

*Реклама со логотипом — с erid и маркировкой «Реклама».*

## KPI пилота 60 дней

| Метрика | Цель |
|---------|------|
| Ученики, открывшие карту по QR | ≥40% нового потока |
| NPS школы (вопрос «помогла ли карта») | ≥45 |
| Жалобы «не смогли заправиться на занятии» | −50% |

## Что нужно от автошколы

1. Размещение QR в 1–3 филиалах (макет даём)  
2. 1 слайд в приветственной презентации ученику  
3. Упоминание в чате учеников (готовый текст — ниже)  
4. Запрет инструкторам рекомендовать «свои» непроверенные чаты вместо карты

### Готовый текст для чата учеников

```
Коллеги, подключили официальную карту наличия топлива — «Карта топлива».
Перед практикой смотрите, где сейчас есть 95: [ссылка]
Отмечайте после заправки — помогает другим. Карта бесплатная.
```

## Почему это выгодно школе

- Меньше сорванных занятий → выше доходимость до экзамена  
- Современный имидж («цифровая автошкола»)  
- Дополнительный контент для соцсетей школы  
- **Нулевые затраты** на пилоте

## Этика

- Школа **не** может платить за отображение «есть бензин» на «своих» партнёрских АЗС  
- В кризис — только бесплатная карта, без спонсорских баннеров на главном экране
