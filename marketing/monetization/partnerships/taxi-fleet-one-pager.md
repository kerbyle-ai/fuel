# Партнёрство: таксопарки и агрегаторы

> **Статус:** черновик one-pager. Пилот — после 10k MAU в городе парка.

## Для кого

- Таксопарки от 30 машин (Яндекс Такси, Ситимобил, местные диспетчерские)
- Агрегаторы с собственным fleet
- Корпоративные перевозчики (аэропорт, вокзал)

## Боль партнёра

| Проблема | Стоимость |
|----------|-----------|
| Водитель 20–40 мин ищет ДТ вместо заказов | Потеря выручки парка, негатив пассажиров |
| Нет единой карты «где сейчас есть 95/ДТ» по городу | Хаос в чатах, устаревшие советы |
| Простой на АЗС с очередью | 1–2 пропущенных заказа / смена |

**Бесплатно для парка (этап пилота):**

## KPI пилота 30 дней

| Метрика | Цель |
|---------|------|
| Водители с установленной картой | ≥60% активного парка |
| Среднее время поиска заправки | −25% (опрос + логи) |
| Жалобы «не нашёл бензин» в чате парка | −30% |
