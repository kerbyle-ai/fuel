# Партнёрство: курьерские службы и доставка

> **Статус:** черновик one-pager. Пилот — город с покрытием 10k+ MAU или плотный краудсорсинг.

## Для кого

- Службы доставки еды (собственный курьерский fleet на авто/мото)
- Курьерские агрегаторы (Самокат, местные dark store)
- Логистика «последняя миля» на легковых авто

## Боль партнёра

| Проблема | Влияние |
|----------|---------|
| Курьер застрял в очереди на АЗС 25+ мин | SLA доставки, штрафы, отток клиентов |
| Нет прогноза «где сейчас быстрее заправиться» | Хаотичные маршруты, перерасход топлива |
| Дефицит ДТ в одном районе — массовый простой fleet | Срыв слотов в пик (обед, вечер) |

## Оффер «Delivery Fuel Intel»

### Для диспетчерской / логиста

- **API или дашборд:** индекс наличия ДТ/95 по районам города (обновление каждые 15–30 мин)
- **Webhook:** «в полигоне [склад/район] <20% АЗС с ДТ» → алерт в Slack/Telegram
- **Маршрут курьера:** виджет «заправка по пути» (интеграция через ссылку / SDK на этапе 50k MAU)

### Для курьера

- PWA / бот — бесплатно  
- Premium «В пути+» — корпоративный пакет: **79 ₽/место/мес** при 50+ лицензиях

## Монетизация

| Пакет | Включено | Цена |
|-------|----------|------|
| **Starter** | Дашборд 1 город, email-алерты | 10 000 ₽/мес |
| **Pro** | API районов, webhook, 5 полигонов | 25 000–45 000 ₽/мес |
| **Enterprise** | SLA 99%, кастомные полигоны, интеграция | от 80 000 ₽/мес |

*Пилот 30 дней: Starter бесплатно в обмен на кейс и NPS от диспетчера.*

## KPI пилота

| Метрика | Цель |
|---------|------|
| Среднее время заправки в смену | −15% |
| Срывы SLA из-за «не нашёл ДТ» | −20% |
| Активные курьеры с картой | ≥70% fleet на авто |

## Техническая интеграция (этапы)

1. **Неделя 1:** UTM-ссылки в приложении курьера + дашборд для логиста  
2. **Неделя 2–4:** сбор метрик, калибровка полигонов  
3. **После 50k MAU:** API `GET /fuel-index?polygon=...` (обезличенные агрегаты)

## Данные и 152-ФЗ

- Передаём только **агрегаты по районам**, не треки курьеров  
- Договор обработки данных — шаблон у юриста  
- Курьерские маршруты **не** продаём третьим лицам

## Этика

- Индекс наличия в кризис — публичный слой остаётся бесплатным  
- Платный API — скорость и полигоны, **не** эксклюзив на правду о дефиците
