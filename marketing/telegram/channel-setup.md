# Настройка Telegram-канала «Карта топлива»

Инструкция для создания официального канала и первого анонса.

---

## 1. Создать канал

1. Telegram → **Новое сообщение** → **Канал**.
2. **Название:** `Карта топлива`
3. **Описание канала** (можно изменить позже):

```
Бесплатная карта АЗС по России — где есть и нет топлива прямо сейчас. Отметки от водителей. Бот @ToplivoBot · Карта в закрепе.
```

4. **Username** (публичная ссылка):
   - `@FuelMapRussia` — рекомендуется
   - `@KartaTopliva` — альтернатива
   - `@toplivo_map` — если заняты

5. Тип: **Публичный канал** (чтобы ссылку можно было шарить).

---

## 2. Аватар канала

Загрузите квадратное изображение **512×512 px** (PNG или JPG).

**Исходник в проекте:**
- `frontend/public/icons/icon.svg` — основная иконка бренда
- `frontend/public/favicon.svg` — упрощённый вариант

Экспорт в PNG (Inkscape, Figma, или скрипт `scripts/generate-icons.py` для готовых размеров из `frontend/public/icons/`).

Telegram: **Настройки канала** → **Изменить** → **Фото канала**.

---

## 3. Добавить бота администратором

1. **Управление каналом** → **Администраторы** → **Добавить администратора**.
2. Найдите `@ToplivoBot` (или ваш username бота).
3. Права на старте:
   - ✅ Публикация сообщений *(если бот будет постить дайджесты)*
   - ✅ Редактирование сообщений *(опционально)*
   - ❌ Остальное — по необходимости

> Для уведомлений о новых отчётах в канал добавьте в `.env`:
> `TELEGRAM_NOTIFY_CHANNEL_ID=-100xxxxxxxxxx`
> ID канала: перешлите любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot) или [@RawDataBot](https://t.me/RawDataBot).

---

## 4. Первый пост и закреп

> **Frontmatter — только для внутреннего учёта, не публикуется.**  
> В файлах `channel-posts/*.md` блок между `---` — метаданные (дата, город, статус).  
> В Telegram уходит только тело поста ниже шапки. Скрипты `scripts/publish-launch-post.mjs` и `scripts/edit-channel-post.mjs` снимают frontmatter автоматически.

1. Скопируйте текст из `marketing/telegram/channel-posts/00-launch-official.md` (**только тело**, без YAML между `---`).
2. Замените плейсхолдеры при необходимости:
   - `@ToplivoBot` → ваш username бота
   - URL карты → актуальный `WEB_APP_URL` или домен
3. Опубликуйте пост.
4. **Закрепить:** долгое нажатие на пост → **Закрепить**.

---

## 5. Описание и ссылки в профиле канала

**Описание** (до 255 символов):

```
⛽ Карта АЗС России — есть/нет топлива от водителей
🤖 Бот: @ToplivoBot
🗺 Карта: http://147.45.175.194:8090
💛 Поддержка: boosty.to/toplivo
```

После появления домена обновите ссылку на карту в описании и закреплённом посте.

---

## 6. Чеклист запуска

- [ ] Канал создан, username выбран
- [ ] Аватар загружен (`frontend/public/icons/icon.svg` → PNG 512×512)
- [ ] Первый пост опубликован (`00-launch-official.md`)
- [ ] Пост закреплён
- [ ] Бот добавлен админом
- [ ] Ссылка на канал добавлена в бота/сайт (по мере готовности)
