# Донаты и поддержка проекта

«Карта топлива» — **бесплатный** краудсourced-сервис: данные о наличии топлива на АЗС добавляют пользователи. Донаты **необязательны** и **не открывают** платный функционал — это добровольная поддержка разработки, как на [pdfuel.ru](https://pdfuel.ru/).

## Как это работает в приложении

В боковой панели показывается ненавязчивый блок:

- текст: «Проект бесплатный. Данные добавляют пользователи.»;
- кнопка «Поддержать проект» → внешняя ссылка на платёжную страницу (новая вкладка).

Бэкенд не нужен: только ссылка, задаётся переменной окружения при сборке фронтенда.

## Настройка

1. Зарегистрируйтесь на платёжном сервисе (рекомендуется **CloudTips**).
2. Создайте публичную ссылку на приём чаевых / донатов.
3. Добавьте URL в `.env`:

```env
VITE_DONATION_URL=https://cloudtips.ru/p/ваш_ник
```

4. **Пересоберите** фронтенд (Vite вшивает переменные на этапе `build`):

```bash
# Локально (dev)
cd frontend && npm run dev

# Docker (dev)
docker compose up -d --build nginx

# Production (VPS)
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build nginx
```

Если `VITE_DONATION_URL` пустой или не задан — баннер скрыт.

## CloudTips (рекомендуется)

Сайт: [cloudtips.ru](https://cloudtips.ru/) — популярная в РФ платформа для чаевых и донатов (аналог того, что использует pdfuel).

### Пошагово

1. **Регистрация** — зайдите на [cloudtips.ru](https://cloudtips.ru/), войдите через банк / телефон / соцсеть (по инструкции на сайте).
2. **Профиль / кошелёк** — привяжите карту или счёт для вывода средств.
3. **Ссылка на чаевые** — в личном кабинете создайте или скопируйте публичную ссылку вида:
   ```
   https://cloudtips.ru/p/ваш_ник
   ```
   или короткую ссылку, которую выдаёт сервис.
4. **Проверка** — откройте ссылку в браузере, убедитесь, что страница оплаты открывается.
5. **Деплой** — вставьте URL в `VITE_DONATION_URL` в `.env` на сервере и пересоберите контейнер `nginx` (см. команды выше).

Комиссию и лимиты смотрите в актуальных тарифах CloudTips.

## Юридические замечания (РФ)

> Не юридическая консультация. При сомнениях обратитесь к бухгалтеру или юристу.

- **Добровольные переводы** («на поддержку проекта», «чаевые») от физлиц обычно не требуют от пользователя оплаты за доступ к карте — у нас нет paywall, это важно сохранять.
- **Получение денег на регулярной основе** может потребовать статус **самозанятого (НПД)** или **ИП**. Лимиты и правила НПД меняются — сверяйтесь с [ФНС](https://npd.nalog.ru/).
- **Чеки**: самозанятые формируют чеки в приложении «Мой налог»; ИП — по своей системе налогообложения. Платформы (CloudTips и др.) часто подсказывают, как оформить выплаты.
- **Не позиционируйте** донат как оплату товара или услуги с обязательным результатом — формулировка «поддержать разработку» / «поблагодарить автора» ближе к добровольным переводам.

## Принципы (как у pdfuel)

- Карта остаётся **полностью бесплатной**.
- Данные — от **сообщества**, не от платной подписки.
- Донат — **по желанию**, без давления и без блокировки функций.
- Никаких платёжных данных на наших серверах — только редирект на внешний сервис.
